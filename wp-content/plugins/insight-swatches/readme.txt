== Description ==

Insight Swatches provides a much nicer way to display variations of variable products. This plugin will help you select style for each attribute as color, image or text.

= Features =

- Style for each attribute as color
- Style for each attribute as image
- Style for each attribute as text

= Need support? =

Visit [plugin documentation website](https://thememove.com "plugin documentation")

== Installation ==

1. Go to plugins in your dashboard and select "Add New"
2. Search for "Insight Swatches", Install & Activate it
3. Go to each attribute and config as you want
